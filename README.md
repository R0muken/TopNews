# TopNews django project
## Build project
#1
`pip install -r requirements.txt`

#2
`cd mysite` 

#3
`python manage.py makemigrations`

#4
`python manage.py migrate`



## Start project

`python manage.py runserver`

Test it out http://localhost:8000

## To create superuser:

`python manage.py createsuperuser`
